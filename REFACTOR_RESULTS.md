### 核心改进

1. **严格筛选逻辑**
   - 区分核心词和通用词
   - 必须包含核心词（地名/主要事物）才能被筛选
   - 防止"攻略"、"推荐"等通用词导致不相关内容混入

2. **智能内容生成**
   - 仅使用与主题相关的特征生成内容
   - 当相关内容不足时，使用通用模板填充
   - 避免不相关的内容（如西双版纳景点）混入千岛湖主题

3. **标题质量提升**
   - 过滤包含不相关地名的标题
   - 完善模板占位符替换
   - 确保生成的标题语法正确

4. **标签相关性**
   - 基于主题类型返回通用流量标签
   - 从热点数据提取标签时进行相关性过滤
   - 添加主题本身的标签

### 使用方法

```bash
# 分析热点数据（支持主题筛选）
python scripts/02_analyze_trending.py --topic "主题关键词"

# 生成笔记（主题驱动）
python scripts/03_generate_posts.py --topic "主题关键词" --count 10

# 一键执行
bash scripts/run_daily_pipeline.sh "主题关键词"
```

### 注意事项

1. **数据依赖**: 系统从 `xhs_post_from_search/jsonl/*.jsonl` 中提取特征
   - 如果数据中没有某主题的相关内容，会使用通用模板生成
   - 建议定期更新原始数据以获取最新的热点内容

2. **关键词扩展**: 内置关键词映射表支持常见主题类型
   - 亲子、酒店、骑行、美食、攻略、旅游、户外、摄影、购物、交通
   - 可在两个脚本的 `KEYWORD_EXPANSION` 字典中扩展

3. **严格筛选**: 默认启用严格筛选模式
   - 确保内容相关性
   - 可能导致某些主题筛选结果较少（属正常现象）
